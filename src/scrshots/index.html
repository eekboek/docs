Na het opstarten kunt u een keuze maken uit reeds aangemaakte administraties.

[% INCLUDE util/image image = "Administratiekeuze.png" %]

U kunt dan ook kiezen om een nieuwe administratie aan te maken.

[% INCLUDE util/image image = "Wizard1.png" %]

Er worden meerdere sjablonen voor verschillende soorten administraties bijgeleverd, alsmede een voorbeeldadministratie om mee te oefenen.

[% INCLUDE util/image image = "Wizard2.png" %]

Een krachtige feature van EekBoek is de mogelijkheid om vanuit overzichten detailgegevens op te roepen. Hiermee kunt u snel hoe een bepaald resultaat in de administratie is ontstaan (‘drill-down’). Neem bijvoorbeeld deze balans.

[% INCLUDE util/image image = "MenuBalans.png" %]

U ziet hier balans gesorteerd op grootboeknummer.

[% INCLUDE util/image image = "Balans.png" %]

Als u nu klikt op 2320 krijgt u het overzicht van alle boekingen op rekening 2320, Postbank.

[% INCLUDE util/image image = "Grootboek.png" %]

Wilt u de details weten van boeking “Afschrift 8”? Klik dan op de link Postbank:8.

[% INCLUDE util/image image = "Journaal.png" %]

U kunt nu weer klikken op een rekeningnummer, maar ook op een relatiecode, b.v. ACME.

[% INCLUDE util/image image = "Debiteurenadministratie.png" %]
