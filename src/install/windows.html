Systeemeisen

Elk modern Microsoft Windows systeem (vanaf Windows98) is in staat om zonder problemen EekBoek te installeren en te draaien.

Helaas heeft Microsoft met de komst van Windows Vista een nieuwe stap gezet om de controle over uw computer verder naar zich toe te trekken. De licentievoorwaarden van Windows Vista geven Microsoft het recht om, onder het voorwendsel van veiligheid, de inhoud van uw computer te bekijken en zo nodig programma's en bestanden te verwijderen.
Wij vinden dit niet verenigbaar met het streven van EekBoek om u juist de volledige en voortdurende zeggenschap over uw eigen gegevens te geven. Wij raden daarom het gebruik van EekBoek op Windows af.
Wilt u desondanks toch EekBoek gebruiken dan vindt u hieronder welke stappen daartoe moeten worden ondernomen.

Om EekBoek te kunnen draaien is Perl nodig. Daartoe dient u ActivePerl versie 5.8.8 of later te installeren. Meer informatie hierover op de ActiveState web site.

Controleer eerst of de versie van Perl toereikend is. Open daartoe een command prompt window en geef de opdracht:

perl --version

De uitvoer begint met de melding

This is perl, v5.8.8 built for ...

De Perl versie moet tenminste 5.8.8 zijn. Recentere versies kunt u installeren vanaf de ActiveState web site.

Let verder op dat veel van de genoemde bestandsnamen zowel hoofd- als kleine letters bevatten, en deze exact zo moeten worden ingegeven!

Installeren met PPM

ActivePerl komt met versie 4 van het installatieprogramma ppm, de Perl Package Manager.

De volgende stappen dienen te worden uitgevoerd:

  1. Start de Perl Package Manager.

    Kunt u deze niet zo snel vinden, gebruik dan Start > Run > Open > ppm .

  2. Open het Preferences window met Ctrl-P en open de ‘Repositories’ tab.

  3. Voeg de EekBoek repository toe.

    Name: EekBoek
    Location: http://www.eekboek.nl/repo/windows

    Er volgt enige activiteit.

  4. Open de ‘Areas’ tab. Kies de area waar u EekBoek wilt installeren. Als u het niet weet, kies dan ‘site’. Noteer de bij deze keuze behorende Prefix, b.v. C:/Perl/site. Druk dan op de OK knop.

  5. In het hoofdscherm kunt u nu de gewenste EekBoek versie kiezen en installeren.

  6. Sluit ppm af.

  7. Om de geïnstalleerde EekBoek te kunnen draaien moet het juiste pad worden toegevoegd aan uw PATH environment variabele. Raadpleeg daarvoor de systeemdocumentatie.

    Het juiste pad is de Prefix welke u eerder noteerde, waarin alle ‘/’ door ‘\’ zijn vervangen, en de tekst ‘/bin’ is toegevoegd. Is bijvoorbeeld de prefix C:/Perl/site dan moet het pad C:\Perl\site\bin worden toegevoegd.

Wilt u in de toekomst een nieuwe versie van EekBoek installeren, dan kan dit eenvoudig door de Perl Package Manager opnieuw te starten en in het hoofdscherm de nieuwe versie van EekBoek te kiezen.

Raadpleeg verder de EekBoek documentatie.
