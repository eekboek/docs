Systeemeisen

EekBoek is geschreven in Perl, en daarom onafhankelijk van het onderliggende besturingssysteem. De minimaal benodigde versie van Perl is 5.8. Wij adviseren om tenminste 5.8.3 te gebruiken, dit in verband met mogelijke Unicode problemen.

Als database worden op dit moment de database systemen PostgreSQL en SQLite ondersteund. Deze systemen zijn beschikbaar voor de meeste platforms.
Ondersteuning voor andere populaire databases is geplanned.

Installatie

  • GNU/Linux systemen
  • Microsoft Windows
  • Bouwen vanaf source
