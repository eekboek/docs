De documentatiebrowser

Alle EekBoek documentatie is te bekijken met de documentatiebrowser. Deze kan worden opgeroepen vanuit de grafische shell met de menukeuze ‘Hulp’.

[% INCLUDE util/image image = "ebwxhelp.png" %]

Het is ook mogelijk de documentatiebrowser los te gebruiken, door de grafische shell op te starten met de parameter ‘--showhelp’:

ebwxshell --showhelp
